🌌 Создание подложки в PhotoShop

  1. Выделение всего рисунка на полотне нажимая CTRL + LBM по иконке предпроссмотра слоя
  2. Сжатие выделения на 1-2 px внутрь принта (или провести каллибровку голов)
  3. Создание "Новый плашечный канал..."
  4. Сохранение в PDF файл

💀 RIP (Raster Image Processor)

  1. Импорт файла для "риповки"
  2. Ожидание загрузки файла
  3. Выбор пресета "720x720"
  4. Выделение файла и запуск RIP
  5. Экспорт файла (расширение .prt)
